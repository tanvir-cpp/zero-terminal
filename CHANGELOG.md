# Changelog

All notable changes to Zero Terminal will be documented here.

## [1.0.0] - 2026-03-08

### Added
- Initial release of Zero Terminal theme
- `zero.omp.json` — Oh My Posh theme with GitHub dark color palette
- `install.ps1` — One-command installer for Windows
- `install.sh` — One-command installer for macOS/Linux
- `uninstall.ps1` — Uninstaller for Windows
- `uninstall.sh` — Uninstaller for macOS/Linux
- Windows Terminal color scheme (`zero-scheme.json`)
- CaskaydiaCove Nerd Font auto-install
- Segments: OS, Path, Git, Python, Node.js, Go, Rust, Execution time, Battery, Time, Exit status
- Transient prompt support
- Windows preview screenshot
